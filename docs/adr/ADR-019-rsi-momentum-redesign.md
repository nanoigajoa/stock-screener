# ADR-019: 모멘텀 채점을 StochRSI/BB%B → RSI 종형 곡선으로 재설계

**날짜:** 2026-05-02  
**상태:** 결정됨

---

## 배경

기존 momentum 카테고리:

```python
momentum_score = (stoch_intensity + bb_intensity) / 2
# stoch_intensity: max(0, (0.3 - k) / 0.3)  — k < 0.3 구간만 양수
# bb_intensity:    max(0, (0.2 - pct_b) / 0.2)  — pct_b < 0.2 구간만 양수
```

### 구조적 모순

- `stoch_intensity > 0` 조건: StochRSI k < 0.3 → **과매도 상태**
- `trend_score > 0` 조건: MA20 > MA60 → **상승 추세**

과매도 상태(k < 0.3)는 보통 주가가 이미 많이 내린 상태다.  
상승 추세(MA20 > MA60)는 주가가 두 이동평균 위에 있는 건강한 상태다.  
→ 두 조건이 동시에 만족하는 경우가 매우 드물어, **두 카테고리를 동시에 활성화할 수 없었다.**

결과: trend가 0.7이면 momentum이 0에 가깝고, momentum이 0.7이면 trend가 0에 가까움.  
→ total score가 항상 한쪽에 쏠려 등급 편향 발생.

## 결정

모멘텀의 주 지표를 **RSI(14)** 로 교체.  
"건강한 상승 모멘텀 구간"(45~65)에서 최고점, 양 끝으로 갈수록 감소하는 종형 곡선 채택.

```python
# RSI 구간별 점수 (55 중심 종형)
if 45 <= rsi <= 65:
    rsi_score = 1.0 - abs(rsi - 55) / 10   # 55 → 1.0, 45/65 → 0.0
elif 35 <= rsi < 45:
    rsi_score = (rsi - 35) / 10 * 0.6       # 35 → 0, 45 → 0.6 (눌림목 회복)
elif 65 < rsi <= 75:
    rsi_score = (75 - rsi) / 10 * 0.4       # 65 → 0.4, 75 → 0 (과매수 진입)
# < 35 or > 75 → 0.0
```

### 구간 설계 근거

| 구간 | 시장 상황 | 점수 방향 |
|------|----------|----------|
| 55 중심 (45~65) | 건강한 상승 모멘텀 | 최고점 1.0 |
| 35~45 | 눌림목 회복 진행 중 | 낮지만 양수 (0~0.6) |
| 65~75 | 과매수 진입, 리스크 증가 | 감소 (0.4~0) |
| < 35, > 75 | 극단 구간 (반전 위험) | 0.0 |

**보조 보너스 (RSI만으로 포착 못하는 신호 보완):**

```python
# StochRSI 방향 보너스: 과매도권에서 반등 시작
if k > k_prev and k_prev < 0.35:
    stoch_bonus = min((k - k_prev) / 0.05, 0.3)

# Z-Score 통계 보너스: 2σ 이하 통계적 과매도
z = (close[-1] - mean_20) / std_20
z_bonus = 0.30 if z < -2.0 else 0.15 if z < -1.0 else 0.0

# 동일 현상 이중 계산 방지: 강한 쪽만 채택
momentum_score = min(rsi_score + max(stoch_bonus, z_bonus), 1.0)
```

Z-Score와 StochRSI 보너스는 동일한 "과매도 상태"를 다른 방법으로 측정한다.  
두 신호가 동시에 발화할 때 더 강한 쪽만 취해 중복 계산을 방지한다.

## 고려한 대안

| 대안 | 기각 이유 |
|------|----------|
| StochRSI 임계값 완화 (0.5 이하) | 여전히 "낮을수록 좋다" 논리 — 상승 추세와 역방향 |
| MACD 히스토그램 강도 | 이미 스크리닝 체크리스트에 존재, 시그널 서비스 중복 |
| RSI 단순 하한 (< 60) | 임계값 이진화 → 58과 62 구분 불가 |

## 결과

- trend(MA 정배열) + momentum(RSI 45~65 건강 구간)이 **동시에 높은 점수 가능**
- STRONG BUY(0.60) 도달 방정식: trend=1.0 × 0.35 + momentum=1.0 × 0.25 = 0.60 ✅
- score 분포가 0~1 전 범위로 확산 (기존 0.3~0.5 집중 → 해소)
